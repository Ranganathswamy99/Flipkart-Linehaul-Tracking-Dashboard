# 🚛 Flipkart Linehaul Daily Tracking Dashboard

## Overview
This Power BI dashboard was developed based on my Flipkart Linehaul operations experience.

The dataset was self-designed and AI-generated based on real operational scenarios. No external data source was used.

## 📊 Coverage
- 4 Operational Zones
- 1500+ Vehicles Visibility
- 5383 Total Trips

## 📈 KPIs Tracked
- Total Trips
- In-Transit Vehicles
- On-Time Performance
- Delayed / Halted Vehicles
- Arrived at Hub

## 📍 Dashboard Features
- Category-wise trip analysis
- Halted vehicles by vendor
- Date-wise in-transit trend
- Vehicle status breakdown
- Halted vehicles geo-location map
- Vendor & hub-level tracking

## 🎯 Business Value
Improves SLA monitoring, vendor performance tracking, and real-time operational visibility for logistics teams.
